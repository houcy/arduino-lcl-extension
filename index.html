Introduction

This Scratch extension lets you interact with the physical world using an Arduino/Genuino 101 board. The Arduino/Genuino 101 features a built in tilt sensor and wireless Bluetooth connectivity (coming soon).

The Arduino 101 extension is built for ScratchX, the Experiemental Extension system for the Scratch programming language developed by the Lifelong Kindergarten Group at the MIT Media Lab.


Arduino (left) and Genuino (right) 101 board

You will need:

Hardware

  • Arduino/Genuino 101 board
  • USB Cable - Type A to B

Software

  • Firefox web browser
  • Scratch Device Plugin
    Select the Mac or Windows plugin for other browsers
  • Arduino IDE
